Getting Started

Discord Messages Dump is a Python package that allows you to download and save message history from Discord channels. This guide will help you get started with the package.

Installation

To install Discord Messages Dump, follow these steps:

# Clone the repository
git clone https://github.com/bobbyiscool123/Discord_messages_dump.git
cd Discord_messages_dump

# Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS

# Install dependencies
pip install -r requirements.txt

# Install in development mode (optional)
pip install -e .

Configuration

Before using Discord Messages Dump, you need to set up your Discord token and channel ID. Create a .env file in the root directory of the project with the following content:

DISCORD_TOKEN="YOUR_DISCORD_TOKEN"
DISCORD_CHANNEL_ID="YOUR_DISCORD_CHANNEL_ID"

Replace YOUR_DISCORD_TOKEN and YOUR_DISCORD_CHANNEL_ID with your actual values.

Note: To obtain your Discord token, follow these steps:
  1. Open Discord in your web browser or desktop app
  2. Press Ctrl+Shift+I (or Cmd+Option+I on macOS) to open developer tools
  3. Go to the 'Network' tab
  4. Make any request on Discord, such as changing channels
  5. Find a request in the Network tab
  6. Look for the 'authorization' header in the request headers
  7. The value of that header is your user token

Basic Usage

You can use Discord Messages Dump in two ways: through the command-line interface or the GUI script.

Command-Line Interface

discord-dump dump --token "YOUR_TOKEN" --channel-id "YOUR_CHANNEL_ID" --format json --output-file messages.json

GUI Script

python Dump.py

The GUI script will open a file dialog to select where to save the output file.

API Reference

Discord Messages Dump provides a Python API that you can use in your own scripts. Here's a brief overview of the main classes:

DiscordApiClient

The DiscordApiClient class handles communication with the Discord API. It provides methods to fetch messages from a channel.

from discord_messages_dump import DiscordApiClient

# Create a client
client = DiscordApiClient(token="YOUR_TOKEN")

# Get messages from a channel
messages = client.get_messages(channel_id="YOUR_CHANNEL_ID", limit=100)

# Print the messages
for message in messages:
    print(f"{message['author']['username']}: {message['content']}")

For more details, see the DiscordApiClient API reference.

MessageProcessor

The MessageProcessor class processes raw message data and formats it into various output formats.

from discord_messages_dump import DiscordApiClient, MessageProcessor

# Get messages
client = DiscordApiClient(token="YOUR_TOKEN")
messages = client.get_messages(channel_id="YOUR_CHANNEL_ID", limit=100)

# Process messages
processor = MessageProcessor(messages)

# Format messages
text_output = processor.format_text()
json_output = processor.format_json()
csv_output = processor.format_csv()
markdown_output = processor.format_markdown()

For more details, see the MessageProcessor API reference.

FileProcessor

The FileProcessor class manages file operations, including opening file dialogs and saving content to files.

from discord_messages_dump import FileProcessor

# Create a file processor
file_processor = FileProcessor()

# Save content to a file
file_processor.save_content("Hello, world!", "output.txt")

# Open a file dialog
output_file = file_processor.open_save_dialog("output.txt", "text")

For more details, see the FileProcessor API reference.

CLI Reference

Discord Messages Dump provides a command-line interface (CLI) that you can use to fetch and save messages from Discord channels.

Commands

The CLI provides the following commands:

dump

Fetch messages from a Discord channel and save them to a file.

discord-dump dump --token "YOUR_TOKEN" --channel-id "YOUR_CHANNEL_ID" --format json --output-file messages.json
Options:
  • --token TEXT - Discord token
  • --channel-id TEXT - Channel ID
  • --format [text|json|csv|markdown] - Output format
  • --output-file TEXT - Output file path
  • --limit INTEGER - Maximum number of messages
  • --no-gui - Disable GUI file dialog
  • --verbose - Enable verbose logging

install-completion

Install command completion for bash/zsh.

discord-dump install-completion
